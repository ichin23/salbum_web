# Stack

## Overview
Vue 3 frontend application using Vite and TypeScript.

## Core Technologies
- **Framework**: Vue 3 (Composition API, `<script setup>`)
- **Build Tool**: Vite
- **Language**: TypeScript
- **Routing**: Vue Router
- **Icons**: `lucide-vue-next`

## Styling
- **CSS**: Tailwind CSS (inferred from utility classes)
- **Variables**: CSS variables for theming (`var(--color-surface-2)`, `var(--color-border)`)

## API Client
- Uses native `fetch` via custom service modules (e.g. `fetchService.ts`) with typed generics.
